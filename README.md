# layout-designer Проект - Мессенджер «Hexlet Chat»
## Описание
Проект онлайн-чата, в котором пользователи могут обмениваться сообщениями и вести список контактов.
## Запуск  локально

```sh
make start
```
## Запуск  линтера

```sh
make lint
```
